策略规则：

- 调仓：月频，月末最后一个交易日收盘后生成信号，下一交易日开盘执行。
- 风控：只有当 588200 ETF 收盘价高于 60 日均线时才持仓，否则清仓转现金。
- 选股：按综合得分选 Top 5，等权持仓。
- 综合得分：`0.7 * zscore(ma_trend_20_60) + 0.3 * zscore(liquidity20)`。
- `ma_trend_20_60`：20 日均线 / 60 日均线 - 1，衡量中短期趋势。
- `liquidity20`：20 日平均成交额的对数，偏向流动性更好的成分股。
- 卖出：调仓时不在新 Top 5 内的股票卖出；若风控关闭，则全部卖出。

## 使用方法

在项目根目录运行：

```powershell
python scripts\optimize_588200_strategy.py --start 2022-10-26 --train-end 2024-12-31 --end 2026-04-10
```

常用参数：

- `--cost-rate 0.001`：单边换手成本，默认 0.10%。
- `--train-end YYYY-MM-DD`：训练期截止日；脚本只用训练期选择最优参数。
- `--output-dir PATH`：结果输出目录。
- `--astock-root PATH`：AStock 数据根目录，默认使用迁移后的 E 盘镜像。

每次复跑后重点看：

- `report_*.md`：策略说明和绩效摘要。
- `summary_*.csv`：所有参数组合的训练期、样本外、全样本指标。
- `best_trades_*.csv`：推荐策略每次调仓的持仓名单。
- `best_config_*.json`：推荐策略参数。

实盘/模拟盘使用步骤：

1. 每个月最后一个交易日收盘后运行脚本。
2. 打开最新输出目录下的 `best_trades_*.csv`。
3. 找到最新一行：
   - 如果 `risk_on=False`，下一交易日开盘清仓或保持空仓。
   - 如果 `risk_on=True`，下一交易日开盘按 `selected` 列中的股票等权买入。
4. 下次调仓时重复上述步骤，卖出不在新名单里的旧持仓。

## 重要限制

- 当前股票池是 2025Q4 静态成分股，不是历史 point-in-time 成分股，仍有幸存者偏差。
- 回测没有完整模拟涨跌停无法成交、停牌、盘口深度、滑点冲击和 A 股 T+1 细节。
- “最优”只表示在训练期风险调整收益较优，并通过当前样本外区间复核；不代表未来一定最优。
- 若后续拿到 588200 历史成分变更数据，应优先升级为 point-in-time 动态股票池。
